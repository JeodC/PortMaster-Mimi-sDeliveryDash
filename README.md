## Installation
Download from:https://caffinate.itch.io/mimis-delivery-dash and extract data.win and both .ogg files to `/ports/mimisdeliverydash/gamedata`.

## Default Gameplay Controls
| Button | Action |
|--|--|
|D-PAD|Move around|
|A|Boost|
|B|Collect items|
